#### Подготовка к установке.
##### Записи в DNS.
Необходимо сделать A и MX записи в виде:
# MX записи
```bash
mx - 10.0.0.1
mx1 - 10.0.0.2
mxN - 10.0.0.N
```
# А записи
```bash
smtp - 192.168.77.9, 192.168.77.13
imap - 192.168.77.9, 192.168.77.13
```
